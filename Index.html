Estrutura do HTML

(Voltar)

Os elementos que formam o conteúdo da página são postos entre a tag de abertura e a tag de fechamento . O elemento é também conhecido como o elemento raiz. O elemento HTML ( ou HTML root element) representa a raiz de um HTML ou XHTML documento.

Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags , e e a instrução .

O que preciso saber sobre CSS

Uma das primeiras coisas que você precisa aprender ao estudar CSS são os seletores e as propriedades. Os seletores são utilizados para identificar os elementos HTML aos quais você deseja aplicar estilos. Já as propriedades são os atributos específicos que definem a aparência dos elementos selecionados.

O que é preciso saber de JavaScript

O que é JavaScript exemplos? O JavaScript é a linguagem que permite criarmos páginas web cada vez mais dinâmicas e interativas, fechando a famosa tríplice HTML, CSS e JavaScript. Além disso, hoje também podemos usar o JavaScript no Back-end graças ao Node.
